## ⚡ Step 1: Install Backend

```bash
cd c:\Users\Lenovo\Downloads\ecommerce\backend
npm install
npm run dev
```

**You should see:**
```
✅ MongoDB connected
🚀 Express server running on http://localhost:5000
```
